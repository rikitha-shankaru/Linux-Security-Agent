About This Project

The Linux Security Agent is an advanced security monitoring system that uses eBPF (Extended Berkeley Packet Filter) to capture system calls at the kernel level and machine learning algorithms to detect anomalies and threats in real-time.

This project was developed as part of a Master's degree program to explore kernel-level security monitoring and machine learning-based threat detection. It demonstrates state-of-the-art techniques in cybersecurity, including unsupervised anomaly detection and behavioral analysis.
